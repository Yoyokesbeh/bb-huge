# Contributing to bb-huge 🤗

## Branch Strategy
- `main` — stable only, no direct pushes
- `dev` — all PRs target here
- `feature/xxx`, `fix/xxx`, `skill/xxx`

## How to Contribute
1. Open describe or issue (describe your updates/changes)
2. wait for further insights and approve
4. Fork → branch off `dev`
5. Make your change
6. Open a PR → `dev` with a clear description

## Commit Format
`feat:` `fix:` `skill:` `docs:`

## Two Layers
- **AI Context Layer(Skill)** → `skills/bb-huge/`
- **Portal** → `app/`, `mcp_server.py`

No credentials hardcoded. Ever.
